Attribute Converter

Table of Content :

Introduction

The attribute conversion facility allows the developer to specify methods to convert between the entity attribute representation and the database representation for attributes of basic types. Converters can be used to convert basic attributes defined by entity classes, mapped superclasses, or embeddable classes.
Converter annotation is not applicable on Id attributes, version attributes, relationship attributes and attributes annotated as Temporal or Enumerated.

Implement Converter :

A Converter must implement the javax.persistence.AttributeConverter<X,Y> interface, where X is the class of the entity representation and Y the class of the database representation of the attribute. A converter implementation class must be annotated with the javax.persistence.Converter annotation.

Converter Scope :

  1. If the value of the autoApply element of the Converter annotation is true, the converter will be applied to all target attributes of the given type.
  2. If the value of the autoApply element of the Converter annotation is false, you need to add the javax.persistence.Convert annotation to all attributes that shall be converted and specify the Converter class.
