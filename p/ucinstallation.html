Here comes a walk-through of how to install the Jeddict Plugin in NetBeans using AutoUpdate center.

Add the new update center to NetBeans :

To get the plugins ready for installation, start NetBeans IDE as described in "Installing and the Starting the NetBeans IDE."

  • Select the update center url based on your local installed NetBeans IDE Build number :
    NetBeans IDE versionJeddict versionUpdate Center URL
  • Start NetBeans and select the "Tools > Plugins" menu item Here you need to navigate to Settings tab Type name e.g. "Jeddict" , Add an address of our copied update center and Finish the wizard : Now the Jeddict is available in Update Center Wizard and you can use it to update modules contained in the update center.

Installing Jeddict Plugins :

  • Once the URL is added navigate to Available Plugins tab in the same dialog. Find Jeddict plugin here and select it for installation.
  • Now select the "Install" button at the bottom :
  • When the NetBeans IDE Installer screen is displayed, click Next.
  • Accept the license terms and click Install.
  • If additional Validation screens appear, including a confirmation to install an unsigned plugin. Just select "Continue" until you get to the final screen.
  • The plug-in has now been installed and can be seen in the "Installed" tab:
  • Click Finish to restart NetBeans IDE.

Verifying Your Installation:

Once you have installed the Jeddict Plugin into NetBeans, its installation can further be confirmed via the "New File > Persistence > JPA Diagram file"; if it shows up, the installation seems to have gone ok: The Jeddict is now ready for you to work with, to create a new project and develop code.
