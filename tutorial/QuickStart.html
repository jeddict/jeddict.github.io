JPA Modeler Quick Start Tutorial

Welcome to JPA Modeler! This tutorial provides a very simple and quick introduction to the JPA Modeler workflow by walking you through the creation of a simple JPA application. Once you are done with this tutorial, you will have a general knowledge of how to create enterprises applications model using JPA Modeler.

This tutorial takes less than 10 minutes to complete.

Setting Up the Project :

To create an IDE project:
  • Start NetBeans IDE.
  • In the IDE, choose File > New Project(Ctrl-Shift-N) from the main menu, as shown in the figure below :
  • In the New Project wizard, expand the Java Web category and select Web Application as shown in the figure below. Then click Next.
  • Enter ShoppingSystem in the Project name text box and specify the Project Location to any directory on your computer and click next (as shown in the figure below):
  • Select the version of Java EE you want to use with your application and the server to which you want to deploy your application and add context path
  • Select the framework, if any required :
  • Click Finish:
  • The project is created and opened in the IDE. You should see the Projects window, which contains a tree view of the components of the project, including source files, libraries that your code depends on, and so on.

Create the JPA Diagram :

  1. Goto File menu > New File > Persistence category.
  2. From the Persistence , select JPA Diagram and click Next.
  3. Type CartModel for the diagram name.
  4. Type eshop for the Location.
  5. Click Finish.
  6. When you click Finish, the IDE creates the JPA Diagram and opens the diagram in the designer window.

Create the Entities :

  1. Drag & Drop the Entities from the Palette window to designer.
  2. Double click on the Entity and rename to Customer.

Add property :

  1. To create @Id attribute, click on the Customer Entity property.
  2. To create @Basic attribute, click on the Customer Entity property.

Create Relationship :

  1. To create relation, create some other entities :
  2. Now, click on the Customer Entity property.
  3. And drag to ProductOrder Entity.
  4. Same as above create relation @ManyToMany between ProductOrder and Product Entity.

Visualize/Modify Database :

  1. Right click on the diagram > Visualize DB :
  2. To change database connection click on the top side of combobox panel :
  3. To override default table name, double click on table name label :
  4. To override default column name, double click on column :

Delete Join Table :

  1. Right click on the JoinTable > Delete JoinTable :
  2. Enter the column name of new foreign key mapping :
  3. New foreign key mapping is created now :

View SQL :

  1. Right click on the db diagram > View SQL :
  2. You may view/execute the generated SQL :

Generate the entity classes :

  1. Click on the JPA diagram and change the source code package location from property panel.
  2. Right click on the diagram > Generate Source Code.
  3. Select the source code project location.
  4. Click on the entities class to view source code.
You now know how to accomplish some of the most common tasks in the JPA Modeler.
