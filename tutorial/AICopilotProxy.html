🤖 AI Assistant: How to install Copilot API Proxy

This document explains how to install and activate Copilot API Proxy.



📚Node.js

To run Copilot API Proxy, you need to have Node.js installed:

Node.js

📎 Copilot API Proxy installation and activation

Install and run the Copilot API Proxy from the command line.

npx copilot-api@latest start

On the first run, you need to activate the Copilot API Proxy in your GitHub account.


https://github.com/login/device

After activation, close Copilot API Proxy and you can start using the Jeddict AI Assistant in COPILOT_PROXY mode.



📎 Jeddict AI Assistant in COPILOT_PROXY mode

After switching Jeddict AI Assistant to COPILOT_PROXY mode, Copilot API Proxy will start automatically. The plugin includes detection of multiple instances of Apache NetBeans. Copilot API Proxy will only start on the instance that was started first. If you close this instance, Proxy will automatically start on the next instance.
