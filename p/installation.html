Here comes a walk-through of how to install the Jeddict Plugin in NetBeans.

Downloading Jeddict Plugin :

To download the Jeddict, Please click here:

Installing Jeddict Plugin :

To get the plugins ready for installation, start NetBeans IDE as described in "Installing and the Starting the NetBeans IDE."
  • Extract the nbm file(s) from Jeddict.zip file.
  • Start NetBeans and select the "Tools > Plugins" menu item Switch to the "Downloaded" tab and select the "Add Plugins..." button to browse to the downloaded nbm file. Browse the nbm files :
  • Once the file has been selected, NetBeans will display module information as follows:
  • Now select the "Install" button at the bottom :
  • When the NetBeans IDE Installer screen is displayed, click Next.
    • Accept the license terms and click Install.
    • If additional Validation screens appear, Just select "Continue" until you get to the final screen.
  • The plug-in has now been installed and can be seen in the "Installed" tab:
  • Click Finish to restart NetBeans IDE.

Verifying Your Installation:

Once you have installed the Jeddict Plugin into NetBeans, its installation can further be confirmed via the "New File > Persistence > JPA Diagram file"; if it shows up, the installation seems to have gone ok: The Jeddict is now ready for you to work with, to create a new project and develop code.




** Note : You don't need to uninstall any existing Jeddict plugins for the new update except for some special case like uninstall instruction available in Jeddict README.md specific to release or getting issue after migration to new NetBeans IDE version.
